Code to find digits in an Integer

Problem Statement: Write a code in any programming language that counts the digits in a number. 


	#include < stdio.h >
	int main() {
	    long long n;
	    int count = 0;
	    printf("Enter an integer: ");
	    scanf("%lld", &n);
	 
	    // iterate until n becomes 0
	    // remove last digit from n in each iteration
	    // increase count by 1 in each iteration
	    while (n != 0) {
	        n /= 10;     // n = n/10
	        ++count;
	    }

	    printf("Number of digits: %d", count);
	}

Sample Input

Enter an integer: 6989

Sample Output

Number of digits: 4

Variables used in the program

count, n

Time Complexity

O(log10(n))
